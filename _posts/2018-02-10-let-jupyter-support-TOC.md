给 Jupyter 添加书签目录（TOC）



## 环境

macOS
Anaconda
Jupyter

## 安装

在 Terminal 中执行：

```
pip install jupyterthemes
```
## 应用

* 在 Terminal 中依次执行：

```
jt -l
jt -t grade3
```

* 重启 jupyter
