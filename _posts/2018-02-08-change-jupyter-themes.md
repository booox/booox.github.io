让 Jupyter 支持 Javascript




## 环境

macOS
Anaconda

## 安装

在 Terminal 中执行：

```
pip install jupyterthemes
```
## 应用

* 在 Terminal 中依次执行：

```
jt -l
jt -t grade3
```

* 重启 jupyter
