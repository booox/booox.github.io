## 最棒的工具一

纯文本博客： Jekyll + Github

Jekyll: Ruby

### 安装

`gem install jekyll`

### 新建站点及开启

```
$ jekyll new myblog
$ cd myblog
$ bundle install
$ jekyll s
```

### 测试

`http://localhost:4000`

### Jekyll 补充

* 在后台运行 `$ jekyll server -B`
* 关闭后台运行的 jekyll ： `$ pkill -f jekyll` 或 `kill -9 $(lsof -i tcp:4000 -t)`
