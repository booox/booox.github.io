## 最棒的工具

纯文本博客： Jekyll + Github

Jekyll: Ruby

### 安装

`gem install jekyll`

### 新建站点及开启

```
$ jekyll new myblog
$ cd myblog
$ bundle install
$ jekyll s
```

### 测试

`http://localhost:4000`
