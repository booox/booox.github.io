在 Anaconda 中用命令行安装 Python2.7.3



# 安装步骤

创建新的虚拟环境

```
$ conda create --name py2.7.3 python=2.7.3
```

激活新的虚拟环境

```
$ conda activate py2.7.3
```

注销虚拟环境

```
$ conda deactivate
```
